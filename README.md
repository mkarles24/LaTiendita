# LaTiendita

Este proyecto fue creado con Angular CLI versión 14.1.0.

Se trata de una e-commerce ficticia (La Tiendita) donde se aprecia la implementación de algunos de herramientas que nos ofrece el framework, tales como: Reactive Forms, Router Navigate, Services, Pipes, Observable,  Oninit, On input… Etc. 

Asimismo en  este proyecto se utilizan una gran cantidad de componentes de Angular Material.

## Development server

Run `ng serve` for a dev server. Navigate to `http://localhost:4200/`. The application will automatically reload if you change any of the source files.
